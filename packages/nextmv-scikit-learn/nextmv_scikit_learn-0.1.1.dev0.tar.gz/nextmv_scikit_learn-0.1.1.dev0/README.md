# Nextmv & scikit-learn Python SDK

Welcome to `nextmv-scikit-learn`, an SDK for integrating scikit-learn with the
Nextmv platform.

## Installation

Requires Python `>=3.9`. Install using `pip`:

```bash
pip install nextmv-scikit-learn
```
